数据库  
##SQL
--四大操作语句
  1、删 DELETE （DELETE FROM 表 WHERE 条件）
  2、增 INSERT  （INSERT INTO 表 （字段列表）VALUES（值列表））
  3、改 UPDATE  (UPDATE 表 SET 字段=值，字段=值... WHERE 条件)
  4、查 SELECT  (SELECT * FROM 表 WHERE 条件)
  ------------------------------------------------------------------------
##字句
WHERE 条件
比如：WHERE age>18||age>18 AND score(name)<60
ORDER 排序
ORDER BY age ASC(从小到大)||DESC(从大到小)  
---------------------------------------------------------------------------
GROUP 聚类（合并）
COUNT() 统计
MIN() 小
MAX() 大
AVG() 平均
SUM() 总和
------------------------------------------------------------------------------
LIMIT 分页 限制输出
语法：LIMIT 10; 只给数据库的前十条数据
      LIMIT 3,8; 从3开始，要八条数据
---------------------------------------------------------------------------
###字句之间的顺序
WHERE-->GROUP-->ORDER-->LIMIT...
